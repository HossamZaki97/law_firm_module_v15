Progress Billing in Odoo

This module allows your users to create progress billing in Odoo and allow them to print progress billing in PDF format.

Main Features:

  • Allow you to set Total Progress Billing Amount on the Analytic Account / Project.
  • Added Progress Billing Title on Invoice Form and Selection of Project and Auto compute Progress Billings Amounts on invoice form.
  • Allow to print progress billing reports so you can send to your customer by email.
