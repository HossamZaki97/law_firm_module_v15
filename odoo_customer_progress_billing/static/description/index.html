Customer Progress Billing for Project in Odoo

Customer Progress Billing using Analytic Account

This module allows your accounting users to create progress billing and allow them to print progress billing in PDF format as shown in below screenshots.

Main Features:

  • Allow your user to set Total Progress Billing Amount on the Analytic Account form view where you can mention the total amount which is going to be invoiced to your customer with progress billing for your project.
  • System adds Progress Billing Title on Invoice Form and allows you to make selections of a project. Based on the selection system, auto compute different kinds of Progress Billings Amounts on invoice form as shown.
  • Allow your user to print progress billing pdf reports so you can send to your customer by email.
  • For more details please check below screenshots and please watch the video.
