Schedule Activity Dashboard in Odoo

Dashboard for Schedule Activity Management by Users and Managers in Odoo

Schedule Activity Management Extension

This app allows you to view scheduled activities in Dashboard and provides Dashboard for Schedule Activity.


Main Features:

  • Dashboard menu for Schedule Activities for Employees and Managers.
  • Viewing schedule activities by different filters.
  • Single place to view and open different filtered scheduled activities from dashboard.
  • Dashboard of scheduled activities with different models as shown in the screenshots below. You will have to configure models under configuration then dashboard will build based on it.
  • For more details please watch the video and check below screenshots.


Menus Available:

  • Schedule Activity
    • Configuration
      • Schedule Activity
    • Dashboard
